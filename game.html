Comment jouer

Une expérience otome inspirée de Bungou Stray Dogs.

Avancer dans l'histoire

  • Clique sur la boîte de dialogue pour avancer.
  • Cliquer pendant le texte le complète instantanément.
  • Utilise Espace / Entrée / → au clavier. ← ou le bouton Revenir pour reculer.
  • Appuie sur A pour le mode Auto.

Choix & Affinité

  • Tes choix influencent l'affinité avec chaque personnage.
  • Les jauges à gauche suivent tes relations en temps réel.
  • Atteindre 100 débloque la fin de ce personnage.

Fins possibles

  • Bonne fin — une affinité à 100. Le Livre se referme.
  • Mauvaise fin — une affinité à -100. Conséquences inconnues.
  • Fin mystérieuse — aucun extrême. L'histoire recommence.

Sauvegarde

  • Clique sur 💾 pour sauvegarder à tout moment.
  • La progression est stockée localement dans ton navigateur.
